# 프로젝트 목표

## 서비스 목적

분산된 IoT 데이터를 통합하고, 프로토콜/형식/저장소가 다른 데이터까지 폭넓게 통합 관리하며, **실시간 모니터링·원격 제어·분석**을 통해 **무중단 서비스를 지원하는 데이터 통합 플랫폼**을 제공합니다.

## 운영 목표

**데이터 수집 → 모니터링 → 알림 발생 → 제어/OTA 처리 → 기사 출동 처리 → 제품 개선**을 통한 **무중단 서비스 지원**

## 핵심 가치

- **데이터 활용도** 30% → 90%+
- **평균 복구 시간** 4시간 → 15분
- **알람 정확도** 30% → 98%
- **현장 출동** 월 120건 → 18건 (자동 해결률 85% 목표)
- **운영비 절감** 40%
